# Library Management System

A comprehensive Python-based Library Management System for managing books, members, and transactions.

### Book Management
- **Add New Books** - Title, author, ISBN, genre, publication year
- **Update Book Information** - Edit existing book details
- **Delete Books** - Remove books from the system
- **Search Books** - Find books by title, author, or ISBN
- **View All Books** - Browse complete book catalog

### Member Management
- **Register Members** - Name, ID, contact information
- **Update Member Details** - Modify member information
- **Delete Members** - Remove members from system
- **View Member History** - Track borrowing history

### Transaction Management
- **Borrow Books** - Check out books to members
- **Return Books** - Process book returns
- **Calculate Fines** - Automatic fine calculation for overdue books
- **Transaction History** - Complete record of all borrow/return activities

### Reporting
- **Overdue Books Report** - List of books past due date
- **Popular Books** - Most borrowed books
- **Member Activity** - Member borrowing statistics
- **Inventory Status** - Available vs. borrowed books
